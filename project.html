Store Employee Toolkit

LaunchCode Capstone Project

Full-stack web application that enables store employees to manage products, create customer orders, view order history with item-level details, track employee shifts, and support customer credit card sign-ups.

Key Features & Contributions

  • Designed and developed a RESTful backend using Spring Boot to manage products, orders, and order items with relational database integration.
  • Built a dynamic React frontend with reusable components to display products, manage cart functionality, and render orders with nested item lists.
  • Implemented API endpoints to retrieve all orders and filter orders by phone number, enabling efficient search and order tracking.
  • Integrated frontend and backend using the Fetch API for asynchronous data retrieval and state updates in React.
  • Designed a responsive user interface using CSS Flexbox and media queries for usability across desktop, tablet, and mobile devices.

Tech Stack

Java, Spring Boot, Spring Data JPA, REST API, MySQL, React.js, JavaScript (ES6), HTML5, CSS3, Fetch API, Postman, Git
