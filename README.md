# Varalakshmi-12.github.io
Project Description:

This project is a personal portfolio website built to showcase my background, technical skills, and projects as an entry-level software developer. The website highlights my professional journey, resume, project work, and contact information in a clean, accessible format suitable for recruiters and interviewers.
The site is built using Jekyll and deployed with GitHub Pages, allowing for a lightweight, static website that is easy to maintain and update. The project demonstrates my ability to work with modern development tools, structure content effectively, and deploy a production-ready application.

Tools & Technologies :

Jekyll,Ruby,Bundler,Git & GitHub,GitHub Pages,HTML5 & CSS3

The portfolio consists of the following pages:

Home
Introduces my background, technical focus, and professional journey, including a personal story that highlights resilience, adaptability, and continuous learning.

Resume
Displays my professional summary, technical skills, work experience, education, and includes a downloadable PDF resume.

Projects
Showcases my project work, including the LaunchCode Store Employee Toolkit, with descriptions, screenshots, tech stack details, and links to live demos and source code.

Contact
Provides my email, LinkedIn profile, and GitHub profile for professional connections.
